# 島津競合AI動向モニタリングレポート 2026-07-05

## 1. 概要

2026-07-04レポート以降、監視対象企業（Thermo Fisher Scientific、Agilent、Danaher/SCIEX/Beckman Coulter/Leica Biosystems/Cepheid、Waters、Bruker、Revvity、Roche Diagnostics、Siemens Healthineers、Philips、GE HealthCare、JEOL、HORIBA、Hitachi High-Tech、Mettler-Toledo、Sartorius、Bio-Rad、Tecan、Hamilton Robotics、Illumina）について、分析機器AI、ラボ自動化、スマートラボ、医用・診断AI、画像/スペクトル解析、フィールドサービスAI、予知保全、顧客サポートAI、AI対応ソフトウェア、クラウド/データ基盤、提携・買収・製品発表を中心に確認した。

本日確認できた範囲では、前回レポート以降の「新規大型AI発表」は限定的である。一方、6月末から継続している重要テーマとして、Rocheの新型シーケンサー「Axelios」、RocheによるPathAI買収、AgilentのxCELLigence RTCA eSight AI、医療画像AI/クラウドソフトウェア、AIを用いた自律ラボ・装置制御の研究動向は、島津AIX・分析計測・医用・サービス・ソフトウェア/データ基盤に対する示唆が大きいため継続監視対象とする。

## 2. 重要トピック

### 2.1 Roche: Axeliosシーケンサー投入によるIllumina対抗（継続重点監視）

- **重要度: High**
- **概要:** Rocheは2026-06-29に次世代シーケンサー「Axelios」を投入し、Illuminaが強い遺伝子シーケンシング市場に対抗する動きを強めている。報道では、AxeliosはDNA/RNAを長めの分子として読み取り、再利用可能なチップ上でデータを生成し、近リアルタイムで解析できる点が特徴とされる。初期は研究用途施設を対象とし、臨床版に先行して研究者コミュニティでアプリケーションを広げる狙いがある。
- **島津への示唆:**
  - 分析装置の競争軸が、ハード単体から「装置＋解析ソフト＋データ基盤＋アプリケーション開発コミュニティ」へ移行している。
  - 近リアルタイム解析は、LC/GC/MS、MALDI、LCMS、ICP、医用画像等でも差別化要素になり得る。
  - AIX推進では、装置データの標準化、クラウド/オンプレ解析基盤、AI解析ワークフロー、ユーザー別アプリケーション開発環境をセットで検討する必要がある。
- **関連部門:** 分析計測、医用、ソフトウェア、データ基盤、AIX推進。
- **情報源:** https://www.reuters.com/legal/litigation/roche-launches-axelios-gene-sequencer-bid-loosen-illuminas-grip-2026-06-29/

### 2.2 RocheによるPathAI買収: デジタル病理・AI診断の垂直統合（継続重点監視）

- **重要度: High**
- **概要:** RocheはAIデジタル病理企業PathAIの買収で合意しており、デジタル病理、AIコンパニオン診断、がん診断ワークフローの自動化を強化している。Roche Diagnosticsへの統合により、診断機器・試薬・AIアルゴリズム・病理ワークフローを一体化する方向性が明確である。
- **島津への示唆:**
  - 医用AIでは、単一AIアルゴリズムではなく、診断ワークフロー全体を押さえる企業が競争優位を持ちやすい。
  - 島津の医用・分析装置でも、画像/スペクトル/検査データを個別解析するだけでなく、診断・研究プロセス全体に組み込む設計が重要。
  - 外部AI企業との提携・買収・共同開発の選択肢を、AIXロードマップ上で明確化する必要がある。
- **関連部門:** 医用、分析計測、ソフトウェア、データ基盤、AIX推進。
- **情報源:** https://www.reuters.com/legal/litigation/switzerlands-roche-agrees-acquire-us-based-pathai-2026-05-07/

### 2.3 Agilent: xCELLigence RTCA eSight AI（AI画像解析・細胞解析）の継続監視

- **重要度: High**
- **概要:** AgilentのxCELLigence RTCA eSight AIは、細胞解析・ライブセルイメージング領域でAI画像解析を組み込む動きとして重要。前回までのレポートでHighとして扱った通り、分析計測機器メーカーが装置・画像・AI解析・ワークフローを一体化し、研究者の解析負荷を下げる方向性を示している。
- **島津への示唆:**
  - 画像解析AIは、医用画像だけでなく、細胞解析、顕微、材料観察、クロマト/質量分析のピーク解析・スペクトル解釈にも横展開できる。
  - 島津AIXでは、分析結果の自動解釈、異常検出、ピーク同定支援、レポート自動生成、実験条件最適化まで含めた「解析AIモジュール群」の整備が必要。
  - 装置販売後のソフトウェア価値・サブスクリプション収益化の観点でも重要。
- **関連部門:** 分析計測、医用、ソフトウェア、AIX推進。
- **情報源:** https://www.agilent.com/about/newsroom

### 2.4 GE HealthCare/Philips/Siemens Healthineers: 医療画像AI・クラウド/ソフトウェア基盤の継続競争

- **重要度: Medium**
- **概要:** GE HealthCare、Philips、Siemens Healthineersは、医療画像AI、クラウドPACS、ワークフロー最適化、読影支援、病院内データ統合を継続的に強化している。GE HealthCareはIntelerad買収によりクラウド型医療画像ソフトウェア/SaaSの強化を進めており、AIアルゴリズムの普及基盤としても重要である。
- **島津への示唆:**
  - 医用事業では、装置性能に加え、画像ワークフロー、クラウド連携、AI読影支援、保守データ活用が競争軸になる。
  - 医療機関向けには、AIそのものよりも、業務時間短縮、読影品質の平準化、データ連携、セキュリティ/規制対応を合わせた提案が必要。
  - フィールドサービスAI、予知保全、リモート診断を医用装置と組み合わせる余地が大きい。
- **関連部門:** 医用、サービス、ソフトウェア、データ基盤、AIX推進。
- **情報源:** https://www.reuters.com/legal/transactional/ge-healthcare-acquire-intelerad-23-billion-2025-11-20/

### 2.5 自律ラボ・LLM装置制御: 競合発表ではないが、分析機器AIへの波及が大きい研究動向

- **重要度: Medium**
- **概要:** LLMを用いた実験装置制御、自律ラボ、AIエージェントによる研究ワークフロー自動化の研究が増えている。2026年の研究では、LLM/AIエージェントが実験装置の制御スクリプト作成や装置操作戦略の反復改善を支援できる可能性が示されている。
- **島津への示唆:**
  - 分析装置の将来像は、ユーザーが細かく条件設定する装置から、目的・制約・サンプル情報を入力するとAIが測定条件、前処理、解析、報告まで支援する装置へ向かう可能性がある。
  - AIX基盤では、装置API、データ標準、監査証跡、GxP/CSV、セキュリティ、モデル評価、Human-in-the-loopを含めた設計が必要。
  - 研究所・品質管理・製造現場向けの「Smart Lab / Autonomous Lab」構想として、島津が早期にPoCテーマを整理すべき領域である。
- **関連部門:** 分析計測、ソフトウェア、サービス、データ基盤、AIX推進。
- **情報源:** https://arxiv.org/abs/2604.03286

## 3. 島津/AIX推進への示唆

### 3.1 AIX推進全体

- 競合のAI活用は、個別ツール導入ではなく、装置、データ、ソフトウェア、クラウド、ワークフロー、保守、顧客接点をつなぐ方向に進んでいる。
- 島津AIXでは、社内業務効率化だけでなく、製品・サービス価値に直結するAIXテーマを優先テーマとして整理する必要がある。
- 特に、分析装置データの標準化、装置ログ/測定データ/解析結果の統合、AI解析モジュール、顧客向けクラウド/オンプレ基盤、サービスAIを横断的に設計することが重要。

### 3.2 分析計測事業

- LC/GC/MS、MALDI、UV、IR、ICP、X線、表面分析などで、AIピーク検出、スペクトル同定、異常検知、測定条件最適化、レポート自動化が競争軸になる。
- Roche Axeliosのように、装置データを近リアルタイムで解析し、ユーザーがアプリケーションを拡張できる仕組みは、分析装置にも応用可能。
- AgilentのAI画像解析の動きは、分析機器メーカーが「測る」だけでなく「解釈する」領域へ踏み込んでいることを示す。

### 3.3 医用事業

- Roche/PathAI、GE HealthCare/Intelerad、Philips、Siemens Healthineersの動きから、医用AIは画像・病理・診断ワークフロー・クラウド基盤の統合競争に入っている。
- 島津医用では、AI読影支援だけでなく、検査前後の業務、装置稼働、保守、レポート、病院内データ連携まで含めた価値提案が必要。

### 3.4 サービス・フィールドサポート

- 競合のAI発表は製品AIに目が行きがちだが、装置ログ、稼働情報、エラー履歴、消耗品情報を用いた予知保全・遠隔支援・サービスナレッジAIも重要な差別化領域である。
- 島津では、サービス部門の過去対応履歴、FAQ、技術文書、装置ログをRAG/AIエージェントで活用するテーマをAIX重点施策に位置づけるべき。

### 3.5 ソフトウェア・データ基盤

- 競合の方向性は、AI機能を単独で売るよりも、装置制御ソフト、解析ソフト、クラウド、SaaS、データプラットフォームに組み込む形が中心である。
- 島津としては、装置横断のデータモデル、API、ID管理、権限管理、監査証跡、セキュリティ、グローバル展開を含む共通基盤設計が必要。

## 5. 本日の結論

本日は、前回レポート以降の新規大型AI発表は限定的であった。ただし、Roche Axelios、Roche/PathAI、AgilentのAI画像解析、GE HealthCare等の医療画像AI/クラウド基盤、自律ラボ研究は、いずれも「装置＋AI＋データ基盤＋ワークフロー」の統合競争を示している。

島津としては、AIXを社内業務効率化に閉じず、分析計測・医用・サービス・ソフトウェアの製品価値向上に直結するテーマとして、以下を早期に具体化することが望ましい。

1. 分析装置データの標準化とAI解析基盤
2. 画像・スペクトル・クロマトデータのAI自動解釈
3. 装置ログを用いた予知保全・サービスAI
4. 顧客向けクラウド/オンプレ解析プラットフォーム
5. Smart Lab / Autonomous Labに向けた装置API・AIエージェントPoC
6. 医用AIにおける画像ワークフロー・診断支援・保守データ活用
