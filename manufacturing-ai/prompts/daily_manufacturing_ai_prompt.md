# Daily Manufacturing AI Use Case Monitor Prompt

毎朝、世界の製造業におけるAI活用事例について前回レポート以降の新しい情報を確認してください。

## 監視テーマ

- 製造現場AI（外観検査、異常検知、予知保全、品質改善）
- 生産計画・需給・SCM・ロジスティクス最適化
- 設計・開発・R&DでのAI活用
- フィールドサービス・保守・顧客サポートでのAI活用
- デジタルツイン、シミュレーション、産業用AIエージェント
- ロボティクス、画像処理、フィジカルAI
- 製造業AI基盤、データ基盤、MLOps、AIガバナンス

## 重点的に確認する企業・領域

- Siemens, Bosch, Schneider Electric, ABB, GE Aerospace, Honeywell
- Hitachi, Mitsubishi Electric, Panasonic, Sony, NEC, Fujitsu
- Siemens Healthineers, Philips, Thermo Fisher, Agilent, Danaher, Waters, Bruker, Roche
- NVIDIA, Microsoft, AWS, Google Cloud, SAP, Dassault Systemes, PTC, Oracle

## 出力条件

1. 公式情報、企業公式ブログ、信頼性の高い一次情報・主要メディアを優先する。
2. 重複ニュースは除外する。
3. 日本語で簡潔に要約する。
4. 島津製作所のAIX推進への示唆を必ず記載する。
5. 各項目について以下を記載する。

- 更新日
- 企業・組織名
- 分類（製造・品質・保守・SCM・R&D・サービス・基盤/ガバナンス）
- 新しい情報の概要
- 島津製作所/AIX推進への示唆
- 重要度（高・中・低）
- 情報源URL

## 出力ファイル

`manufacturing-ai/reports/YYYY-MM-DD.md`
