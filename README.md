# regulation-monitor

Regulation monitoring repository for tracking updates related to:

- RoHS
- Battery Regulation
- CBAM
- AI Act
- Cyber Resilience Act

## Purpose

This repository stores daily monitoring notes, source links, and Japanese summaries of regulatory updates.

## Folder structure

- reports/ : Daily and weekly Japanese reports
- sources/ : Source links and raw notes
- prompts/ : Prompts for Codex and ChatGPT
- scripts/ : Future automation scripts
