# Daily Global AI Trend Monitor Prompt

毎朝、世界のAI動向について前回レポート以降の新しい情報を確認してください。

## 監視テーマ

- 生成AI・基盤モデルの新サービス、新モデル、新機能
- AIエージェント、業務自動化、RAG、マルチモーダルAI
- 製造業・医療・分析機器・研究開発領域でのAI活用事例
- AIガバナンス、AI安全性、AI規制、AIリスク管理
- 主要企業動向（OpenAI, Google DeepMind, Microsoft, Anthropic, Meta, NVIDIA, AWS, Apple, Mistral 等）
- 日本企業・日系製造業・競合企業のAI活用動向

## 出力条件

1. 公式情報、企業公式ブログ、信頼性の高い一次情報・主要メディアを優先する。
2. 重複ニュースは除外する。
3. 日本語で簡潔に要約する。
4. 島津製作所・DX戦略・AIX推進への示唆を必ず記載する。
5. 各項目について以下を記載する。

- 更新日
- 分類（技術・製品・事例・規制/ガバナンス・市場/企業動向）
- 新しい情報の概要
- 島津製作所/AIX推進への示唆
- 重要度（高・中・低）
- 情報源URL

## 出力ファイル

`ai-trends/reports/YYYY-MM-DD.md`
